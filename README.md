# App Analytics Dashboard

A simple analytics dashboard built with Spring Boot, Bootstrap, and Chart.js. It displays user activity, screen views, event types, and top screens in a clean responsive interface.

## Features

- Daily Active Users chart
- Top Screens bar chart
- Event Types donut chart
- Summary KPI cards
- Time filters: Last 7 Days, Last 30 Days, All Time
- Responsive layout for desktop and mobile

## Tech Stack

- Java
- Spring Boot
- Bootstrap 5
- Chart.js
- HTML, CSS, JavaScript

## How It Works

The dashboard loads analytics data from REST API endpoints and renders it in summary cards and charts. The filter buttons update the displayed data based on the selected time range.

## Project Goal

This project was created to present app analytics in a clear, simple, and interactive way.
